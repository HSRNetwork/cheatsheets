# Windows Commands
## Interface
_Shows local ip addresses:_
```cisco
ipconfig
```

_Shows detailed local ip addresses:_
```cisco
ipconfig /all
```

_Shows the current routing table:_
```cisco
route print
```

_Shows the current IPv6 routing table:_
```cisco
route print -6
```
